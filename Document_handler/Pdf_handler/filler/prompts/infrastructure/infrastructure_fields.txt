Tu vas recevoir un document administratif structuré en JSON.

Voici le document source :

{{data}}

Ta tâche est d'extraire les informations suivantes, si elles sont disponibles dans le contenu :
- adresse : adresse de l’établissement (ou "")
- transports : un objet contenant :
    - bus : liste des lignes de bus (ou [])
    - metro : liste des lignes de métro (ou [])
    - station : nom de la station la plus proche (ou "")

Si une information est absente ou non identifiable, mets la valeur à `""`.

Réponds strictement avec un JSON au format suivant :

{
    "adresse": "...",
    "transports": {
      "bus": ["..."],
      "metro": ["..."],
      "station": "..."
    }
}
